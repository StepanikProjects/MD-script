prerelease version:
Друзья! Спешим поздравить вас с наступившей зимой!
Спустя много времени после последнего обновления (19.11.17) успело пройти немало работ по улучшению и развитию структуры скрипта.
- Было добавлено множество слов для функции "автораскладка". 
- Возвращена функция выбора игрока, для проведения собеседования (ПКМ + G).
- Добавлено диалоговое меню при выборе игрока (ПКМ + X) для дальнейшего взаимодействия с ним ( для каждого ранга разные функции ). 
- Немного переработан интерфейс скрипта для его стабильности и плавности.
- Проработана и добавлена функция проверки отсутствующих игроков при построении.
- Добавлены автоматические посты для армии Лос-Сантоса и Сан-Фиерро. Лидерам просьба отписаться ВКонтакте, если названия указаны неверно. Нажмите на кнопку "тех. поддержка", мы будем там!
- Произведены некоторые работы над оптимизацией скрипта, чтобы убрать CRASH-ошибки.
- Были сделаны новые картинки для стандартного окна скрипта «в зимнем варианте».
- Добавлены две фунцкии, которые будут оповещать вас о ново-вышедших обновлениях, чтобы вы знали о валидности вашей версии:
	* реклама в чате.
	* окошко "msgbox", которое оповещало бы вас о нем.
- Внесены команды в справку по кнопке F1 и ALT+F1.
- При прописывании команды /time теперь будет отыгрываться то, как Вы смотрите на часы и в /do отображается время сервера.
- При нажатии комбинации клавиш при просмотре времени будет прописываться время в /do строке.
- Исправлена ошибка в отыгровке команды "/lock" >> при женском поле писалось "нажал на кнопку" вместо "нажала".
И прочие незначительные нововведения.
